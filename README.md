# Business Reporting Dashboard

A web-based internal reporting and dashboard application developed to support operational reporting, KPI tracking, and data-driven decision-making.

This application was built to centralise reporting processes, improve visibility into organisational performance, and reduce reliance on manual spreadsheet-based reporting workflows.

---

## Project Overview

The Business Reporting Dashboard enables users to view, manage, and analyse operational data through an accessible reporting interface.

The system was designed to support business users by providing summarized reporting, performance insights, and improved access to operational metrics.

The project demonstrates practical application of:

- Full-stack web development
- Reporting systems development
- Data presentation & dashboarding
- Backend business logic implementation
- Business technology problem solving

---

## Installation

Clone the repository:

```bash
git clone https://github.com/PalesaKM/business-reporting-dashboard.git
```

Navigate into the project directory:

```bash
cd business-reporting-dashboard
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start the development server:

```bash
python manage.py runserver
```
